# Axela

Personal digest bot that aggregates updates from work tools and sends summaries via Telegram.

## Features

- Collect updates from multiple sources (GitHub, Jira, Slack, etc.)
- AI-powered summarization via AWS Bedrock
- Scheduled digest delivery via Telegram
- Web UI for configuration

## Quick Start

```bash
make dev    # Install dependencies
make run    # Run locally
```
